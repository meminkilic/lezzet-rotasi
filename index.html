Proxy sunucu adresi

Gerçek Google verisi için app.py proxy sunucusunu çalıştır, sonra adresini yaz. API anahtarın proxy'de durur, bu dosyada görünmez.

Bilgisayardan: http://localhost:5000 · Telefondan (Wi-Fi): http://192.168.1.X:5000

Proxy nasıl çalıştırılır?

1) pip install flask flask-cors requests gunicorn
2) set GOOGLE_API_KEY=AIza...
3) python app.py
4) Adresi yukarı yaz, test et.

Google Places API — adım adım

Google Haritalar verisini kullanmak için ücretsiz başlayan bir anahtar gerekir.

1

Google Cloud Console

console.cloud.google.com adresine Google hesabınla giriş yap.

2

Yeni proje oluştur

Üst menüden proje seçici → Yeni Proje → isim ver.

3

Faturalandırmayı aç

Sol menü → Faturalandırma. Kart istenir ama ücretsiz kredi verilir.

4

Places API'yi etkinleştir

API'ler ve Hizmetler → Kitaplık → "Places API (New)" ara → Etkinleştir.

5

Anahtar oluştur

Kimlik Bilgileri → API anahtarı. Çıkan AIza... anahtarı kopyala.

6

Render'a yükle

Render → Environment Variables → GOOGLE_API_KEY = anahtarın.

⚠️ Anahtarını Google Cloud'da sadece Places API'ye kısıtla (restrict). Kötüye kullanımı önler.
