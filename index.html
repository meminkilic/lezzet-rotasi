Proxy sunucu adresi

Gerçek Google verisi için app.py proxy sunucusunu çalıştır, sonra adresini buraya yaz. API anahtarın proxy'de durur, bu uygulamada veya paylaşılan dosyada hiç görünmez.

Bilgisayardan: http://localhost:5000
Telefondan (aynı Wi-Fi): http://192.168.1.X:5000 — bilgisayarının IP'sini ipconfig ile öğren. Adres bu cihazın yerel hafızasında tutulur, boş bırakırsan örnek veri modu çalışır.

Proxy nasıl çalıştırılır?

1) app.py dosyasını bilgisayara al.
2) pip install flask flask-cors requests
3) Anahtarı tanımla: setx GOOGLE_API_KEY "AIza..." (pencereyi yeniden aç)
4) python app.py
5) Adresi yukarı yaz, "Test et"e bas.

Google Places API anahtarı — adım adım

Google Haritalar verisini kullanmak için ücretsiz başlayan bir anahtar gerekir. Aşağıdaki adımları sırayla izle.

1

Google Cloud Console'a gir

console.cloud.google.com adresine Google hesabınla giriş yap.

2

Yeni proje oluştur

Üst menüden proje seçiciye tıkla, Yeni Proje de ve bir isim ver (örn. "Lezzet Rotası").

3

Faturalandırmayı aç

Sol menü → Faturalandırma. Kredi kartı istenir ama Google aylık ücretsiz kullanım kredisi verir. Düşük kullanımda ücret çıkmaz.

4

Places API'yi etkinleştir

Sol menü → API'ler ve Hizmetler → Kitaplık. "Places API" ara ve Etkinleştir'e bas.

5

Anahtarı oluştur

API'ler ve Hizmetler → Kimlik Bilgileri → Kimlik bilgisi oluştur → API anahtarı. Çıkan AIza... anahtarını kopyala.

6

Uygulamaya yapıştır

"API Ayarı" sekmesine geç, anahtarı yapıştır, Test et'e bas. Yeşil yanarsa hazırsın.

⚠️ Güvenlik: Anahtarını Google Cloud'da "kısıtla" (restrict). Sadece Places API'ye izin ver ve mümkünse adres/uygulama kısıtı koy. Böylece anahtarın kötüye kullanılmaz.
